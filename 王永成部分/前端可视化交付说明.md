## 1 交付目标

本交付说明用于记录前端可视化模块的实现内容。前端页面用于课程答辩演示，覆盖系统首页、趋势分析、信息搜索、数据源管理和报表中心，能够通过 REST API 调用后端服务，展示信息收集整合系统的主要业务闭环。

## 3 页面功能

| 页面 | 功能 | 对应 API |
|------|------|----------|
| 系统首页 | 服务状态、热点主题数、信息总数、数据源数、热点榜、热度分布 | `/api/v1/health`, `/api/v1/trends`, `/api/v1/items`, `/api/v1/sources` |
| 趋势分析 | 展示主题、分数、趋势方向、统计周期和主题详情 | `/api/v1/trends`, `/api/v1/trends/{topic}` |
| 信息搜索 | 按关键词、来源 ID、数量限制查询信息明细，并查看详情弹窗 | `/api/v1/items`, `/api/v1/items/{id}` |
| 数据源管理 | 新增数据源、查看数据源列表、触发采集任务 | `/api/v1/sources`, `/api/v1/tasks/collect` |
| 报表中心 | 生成 summary/detail 的 XLSX/PDF 报表、查看记录并下载文件 | `/api/v1/reports`, `/api/v1/reports/{id}/download` |

---

## 4 运行方式

1. 启动后端服务：

```bash
python3 -m app.main --host 127.0.0.1 --port 8000
```

2. 使用浏览器打开：

```text
frontend/index.html
```

3. 页面默认连接：

```text
http://127.0.0.1:8000
```

若后端端口变化，可在页面右上角 API 输入框修改地址并保存。
