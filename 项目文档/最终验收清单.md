## 2 代码验收

| 模块 | 文件 | 验收结果 |
|------|------|:--------:|
| 启动入口 | `app/main.py` | 通过 |
| API 路由 | `app/api.py` | 通过 |
| 数据模型 | `app/schemas.py` | 通过 |
| 存储与采集 | `app/storage.py` | 通过 |
| 清洗与趋势分析 | `app/analysis.py` | 通过 |
| 前端演示页 | `frontend/index.html` | 通过 |

关键能力：

1. 支持 `news/forum/api/csv` 四类数据源。
2. 支持 `manual/hourly/daily/weekly` 调度字段。
3. 支持 `enabled/disabled` 数据源状态校验。
4. 支持采集任务、原始数据、清洗数据、趋势结果、报表记录持久化。
5. 支持 `surge/up/stable/down/plunge` 五类趋势方向。
6. 支持信息详情、主题详情、XLSX/PDF 报表生成和下载。
7. 支持 10000 条演示数据下的检索性能验证。
